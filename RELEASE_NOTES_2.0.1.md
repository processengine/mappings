# Release notes — 2.0.1

`2.0.1` is a documentation and examples maturity release for the canonical `2.x` line of `@processengine/mappings`.

## Highlights

- README expanded into a full entry point for the library.
- `SPEC.md` and `SPEC_RU.md` expanded into normative library specifications.
- Added a repository example set covering validation, preparation, execution, typed errors, trace modes, prepared artifact usage, ProcessEngine boundary usage, and limitations.
- Documentation terminology aligned across the package.
- Added README badges for npm version, CI status, Node requirement, and license.

## Summary

This release does not change the canonical API or package shape introduced in `2.0.0`.
It strengthens the documentation package so that `@processengine/mappings` reads as a mature standalone ProcessEngine library rather than as a lightly documented runtime.
